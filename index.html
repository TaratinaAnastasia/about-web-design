HTML

HTML (HyperText Markup Language) — стандартизована мова розмітки документів для перегляду вебсторінок у браузері.

Детальніше

CSS

CSS (Cascading Style Sheets) — мова таблиць стилів, яка використовується для опису зовнішнього вигляду HTML-документів.

Детальніше

JS

JavaScript — мова програмування, що дозволяє створювати інтерактивні елементи на вебсторінках.
